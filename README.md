

Install grunt. No prizes here.

```bash
npm install -g grunt-cli
```

```bash
npm install
```

Build the things!

```bash
grunt init
```

Minify that shit for production?

```bash
grunt prod
```

Start your engines.

```bash
npm start

## running production? Add --production
```
